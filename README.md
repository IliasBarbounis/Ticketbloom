# Ticketbloom

<ol>
  <li>Download the files</li>
  <li>Open a Terminal in the directory of the project</li>
  <li>node ./server.js</li>
  <li>Open a web browser and in the url put “http://127.0.0.1:3000”</li>
</ol>
